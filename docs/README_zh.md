AutoAvoider 是一个“仿真优先”的双目视觉避障项目，面向可扩展的数据采集、模型训练与小车安全部署。目标是建立一条可复用的流水线：从虚拟环境出发，最终稳定迁移到真实场景，并具备良好的泛化能力。

**English Doc**: [README](../README.md)

## 核心目标
1. 构建物理仿真环境与双目相机模型。
2. 高质量采集双目 + 控制数据，用于避障学习。
3. 训练并评估具备跨场景泛化的模型。
4. 打通从仿真到实车的控制闭环。

## 仓库结构（规划中）
- `docs/` 项目文档、设计说明与使用指南。
- `configs/` 训练、仿真、部署相关配置。
- `scripts/` 一键脚本（采集/训练/评估）。
- `data/` 数据集目录（不进 git）。
- `sim/` 仿真环境、场景与传感器。
- `perception/` 模型、训练与评估。
- `control/` 规划决策与控制逻辑。
- `vehicle/` 真车适配层与驱动。
- `services/` 接口服务与控制服务（如需要）。
- `web/` 可选的 UI 或面板。
- `tools/` 数据可视化与处理工具。
- `tests/` 测试。

## 路线图
1. 搭建仿真骨架与双目相机模型。
2. 定义数据格式与采集管线。
3. 建立基线避障模型。
4. 评估并迭代 sim-to-real 泛化能力。
